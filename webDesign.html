What is Web Design?


br

Web design refers to the design of websites that are displayed on the internet.


It usually refers to the user experience aspects of website development rather than software development


To be a good web designer, you need to master the craft of enhancing user experience (UX) and user interface (UI). You need to learn how to use colour

grid systems, and web fonts to design websites that are visually appealing and which are responsive to all kinds of devices
